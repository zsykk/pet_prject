This repository contains the code and data for my personal pet project,
where I analyze my own daily step counts over time.

## Data
- Data was collected personally using a step-tracking device.
- No third-party or external datasets were used.

## Structure
- /data: raw and processed data
- /notebooks: analysis notebooks
